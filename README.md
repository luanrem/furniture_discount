## :electric_plug: Installation

1. Clone the repo
   ```sh
    git clone https://github.com/luanrem/furniture_discount
   ```
2. Enter on the folder
  ```sh
    cd furniture_discount
   ```
3. Install YARN or NPM packages
   ```sh
   yarn
   // or
   npm install
   ```
4. Run the project
   ```sh
   yarn dev
   // or
   npm run dev
   ```
